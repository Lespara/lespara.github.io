Diseñando con código

Chris Lee

Algunos! diseñadores tienen miedo de tocar el código y otros simplemente prefieren otras herramientas. Sin embargo, muchos de los que lo prueban descubren que las tecnologías web no son solo para implementar aplicaciones de calidad de producción: son inmensamente poderosas en las etapas de diseño, ya sea para construir algo desde cero o para crear un prototipo de una visión existente.

Como un hacker autoproclamado, tal vez comiences con una ventaja sobre esto. Ponga una comprensión sólida de diseño y código juntos y es posible que encuentre que cada habilidad amplifica la otra.


Una guía práctica para HTML y CSS

Tómese! unos minutos para actualizar sus conocimientos de HTML y CSS o, si no tiene experiencia en desarrollo web, obtenga algunas habilidades básicas. Conocer las tecnologías web es increíblemente útil para diseñar y crear prototipos, incluso si es la plataforma final, si no la web. Asegúrese de conocer al menos las habilidades discutidas en las lecciones 1-7 de la guía para principiantes.

Hormiga

Asegúrese! de descargar el plugin de Emmet (anteriormente conocido como Zen Coding) para su editor de texto favorito (si no tiene uno, pruebe Sublime Text), que le permitirá escribir HTML y CSS a la velocidad del rayo. Lea la documentación y descubra los accesos directos de Emmet para crear una plantilla de documento HTML, elementos HTML y reglas CSS.

Luego intente crear un nuevo proyecto de diseño desde cero utilizando el código. Tal vez es hora de una nueva cartera! Primero crea la estructura HTML y complétala con algún contenido. ¡Ajústelo con CSS básico y luego repita! Construir un diseño en código hace que sea increíblemente fácil de iterar simplemente modificando los valores. A medida que agrega más contenido, sus contenedores se expandirán para adaptarse. No se puede hacer eso con Photoshop!

Descubra DevTools

Cualquier! buen artesano conoce las herramientas a su disposición. Los diversos navegadores populares ofrecen su propia versión de herramientas de desarrollador, que son inmensamente útiles para diseñar con código. Soy un poco parcial, pero prefiero Chrome, y resulta que tiene un gran conjunto de herramientas de desarrollo. Pruebe este curso de video rápido para aprender a inspeccionar los elementos DOM (los elementos de la página web que genera su HTML) y manipular CSS con las herramientas de desarrollo. Si no ha intentado algo así antes, espere hasta que vea lo que estas herramientas pueden hacer y se sorprenderá.

Hoja de trucos de animación CSS3

El diseño! de la interfaz de usuario no se trata solo de cómo se ve algo, sino también de cómo se siente. Las animaciones pueden ser una gran ayuda para crear cierta sensación con una IU. Con CSS transforma las animaciones, ahora estamos mejor capacitados que nunca para diseñar y crear prototipos con el movimiento en mente. Puede obtener más información sobre transformaciones y animaciones en la mitad avanzada del enlace n. ° 1, y luego visitar este enlace para ponerse a pensar en las aplicaciones creativas de ellos. Con gran poder viene una gran responsabilidad: vuelva a verificar antes de usar animaciones.

AngularJS

Una vez! que tenga un buen manejo de HTML y CSS, el siguiente paso natural es aprender algo de JavaScript para agregar datos reales e interactividad a sus prototipos de IU. Es posible que ya entienda JavaScript; si no, pruebe uno de los muchos buenos tutoriales que existen. ¡Después, dale una oportunidad a AngularJS! Es una biblioteca de JavaScript creada para aplicaciones web de grado de producción, pero funciones como la vinculación bidireccional entre HTML y datos la convierten en una herramienta maravillosa para la creación de prototipos. Cuanto más funcional es un prototipo, mejor puede transmitir una visión de diseño a un compañero de equipo; mejor aún, ¡el código de prototipo suficientemente bueno puede establecer una base para una implementación real! Intente diseñar una interfaz de usuario de aplicación agradable en el código y use unas pocas líneas de código angular para implementar realmente la capacidad de crear nuevas tareas pendientes.
